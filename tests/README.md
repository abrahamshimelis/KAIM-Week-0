# Unit Tests for Scripts Modules

This project contains unit tests for the Exploratory Data Analysis (EDA) modules. Each test class corresponds to a module class and tests its methods.

## Tests

1. **TestSummaryStatistics**: This test class tests the `SummaryStatistics` class.

2. **TestDataQualityCheck**: This test class tests the `DataQualityCheck` class.

3. **TestTimeSeriesAnalysis**: This test class tests the `TimeSeriesAnalysis` class.

4. **TestHistograms**: This test class tests the `Histograms` class.

5. **TestBoxPlots**: This test class tests the `BoxPlots` class.

6. **TestScatterPlots**: This test class tests the `ScatterPlots` class.

7. **TestDataCleaning**: This test class tests the `DataCleaning` class.

## Usage

To run these tests, you need to use Python's `unittest` module. Here's how you can do it:

1. Open your terminal or command prompt.
2. Navigate to the project's root directory
3. Run the following command:

```bash
python -m unittest discover -s tests
